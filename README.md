# ROBOTBACKUP

A robust, full-stack autonomous robot navigation and control system implemented in **Modern C++**. This project adheres to industrial software engineering standards, utilizing a modular, decoupled architecture to achieve low-latency and highly reliable closed-loop control on edge computing platforms.

---

## 📂 System Architecture

* **`src/bringup/`**
* **`src/hardware/`**
* **`src/control/`**
* **`src/decision/`**
* **`src/perception/`**
